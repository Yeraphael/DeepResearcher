# DeepResearcher 项目快速启动手册

## 1. 项目结构

```text
DeepResearcher/
├─ backend/    # Python + FastAPI 后端
├─ frontend/   # Vue 3 + Vite 前端
├─ README.md
└─ 项目快速启动手册.md
```

## 2. 环境要求

- Python `3.10+`
- Node.js `18+`
- npm
- `uv`

## 3. 后端启动

### 3.1 进入目录

```powershell
cd F:\Project\agent\DeepResearcher\backend
```

### 3.2 准备配置

首次启动建议：

```powershell
Copy-Item .env.example .env
```

后端现在会自动加载 [backend/.env](F:\Project\agent\DeepResearcher\backend\.env)。

### 3.3 最小可用配置

```env
SEARCH_API=duckduckgo

LLM_PROVIDER=custom
LLM_MODEL_ID=你的模型名
LLM_API_KEY=你的真实密钥
LLM_BASE_URL=你的兼容接口地址
```

注意：

- `SEARCH_API` 填搜索后端名称，不是 API Key
- `TAVILY_API_KEY`、`PERPLEXITY_API_KEY` 这类密钥要单独写
- `LLM_MODEL_ID` 必须是平台真实模型名
- `LLM_API_KEY` 不能保留示例占位值

### 3.4 安装依赖

```powershell
uv sync
```

如果本机 `uv` 全局缓存有问题，可以临时改到项目目录：

```powershell
$env:UV_CACHE_DIR="F:\Project\agent\DeepResearcher\backend\.uv-cache"
uv sync
```

### 3.5 启动服务

```powershell
uv run python src/main.py
```

默认地址：

- 健康检查：[http://localhost:8000/healthz](http://localhost:8000/healthz)
- API 地址：[http://localhost:8000](http://localhost:8000)

## 4. 前端启动

### 4.1 进入目录

```powershell
cd F:\Project\agent\DeepResearcher\frontend
```

### 4.2 检查接口地址

[frontend/.env.local](F:\Project\agent\DeepResearcher\frontend\.env.local) 默认是：

```env
VITE_API_BASE_URL=http://localhost:8000
```

### 4.3 安装并启动

```powershell
npm install
npm run dev
```

默认访问地址：

- [http://localhost:5173](http://localhost:5173)

## 5. 推荐联调顺序

1. 启动后端
2. 访问健康检查接口确认返回 `{"status":"ok"}`
3. 启动前端
4. 在页面输入主题发起研究

## 6. 本次已修复的关键问题

### 6.1 `.env` 不生效

- 原因：后端之前没有自动加载 `backend/.env`
- 现状：已修复

### 6.2 Windows 下搜索工具导入时报编码错误

- 原因：三方库打印 Unicode 日志时触发 `gbk` 编码异常
- 现状：已修复

### 6.3 前端失败后仍显示“研究流程完成”

- 原因：前端只有 `loading` 状态，没有单独维护失败态
- 现状：已修复

## 7. 常见排查

### 7.1 401 `invalid_api_key`

说明：

- 这是外部模型平台鉴权失败，不是模型回退 bug

建议检查：

1. `LLM_API_KEY`
2. `LLM_BASE_URL`
3. `LLM_MODEL_ID`
4. 账号侧模型权限

### 7.2 404 `model not found`

说明：

- 当前 `LLM_MODEL_ID` 在目标平台不可用

建议检查：

1. 模型名是否拼写正确
2. 平台是否支持该模型
3. 当前账号是否开通该模型

### 7.3 Tavily 配置错误

正确示例：

```env
SEARCH_API=tavily
TAVILY_API_KEY=tvly-xxxx
```

错误示例：

```env
SEARCH_API=tvly-xxxx
TAVILY_API_KEY=='tvly-xxxx'
```
